API Documentation

Use the following endpoints to retrieve geolocation data (optional API key: ?key=[your-api-key]). Contact support or register at support@ip-api-mvtp.onrender.com to obtain an API key. Rate limit: 100 requests per IP per 15 minutes.

  • /json/[ip] - JSON format (e.g., /json/8.8.8.8)
  • /xml/[ip] - XML format (e.g., /xml/8.8.8.8)
  • /csv/[ip] - CSV format (e.g., /csv/8.8.8.8)
