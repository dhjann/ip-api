Welcome

Free IP Geolocation API to determine geolocation from IP addresses. Supports JSON, XML, and CSV formats.

API Documentation

Use the following endpoints to retrieve geolocation data:

  • /json/[ip] - JSON format (e.g., /json/8.8.8.8)
  • /xml/[ip] - XML format (e.g., /xml/8.8.8.8)
  • /csv/[ip] - CSV format (e.g., /csv/8.8.8.8)
